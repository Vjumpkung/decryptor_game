# เกมถอดรหัส

เกมนี้ใช้สำหรับค่าย Let Me Tired #2 ในส่วนของกิจกรรมฐาน

<hr>

## วิธีการใช้งานโปรแกรม

```
python main.py [group number (1 - 10)]
```

กรอก เลขกลุ่มหลังจาก main.py
